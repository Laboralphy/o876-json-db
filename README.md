# o876-json-db

*A lightweight, 100% TypeScript JSON document database with indexing support*

A minimalist, in-memory and file JSON database for Node.js/TypeScript applications. It stores documents in collections, 
supports top-level property indexing, and requires minimal external dependencies.


---

## Features
- **Pure TypeScript**: No JavaScript compromises—designed for type safety.
- **Collections**: Organize JSON documents into named collections.
- **Indexing**: Declare indexes on top-level properties for faster lookups.
- **No ODM**: Direct JSON access—no schema migrations or hidden magic.
